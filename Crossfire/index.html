Installation

git clone https://github.com/BCAS-Team/CrossFire.git
cd CrossFire/CrossFireL

Make crossfire.py executable and optionally add it to your system PATH:

# Linux/macOS
chmod +x crossfire.py
sudo ln -s $(pwd)/crossfire.py /usr/local/bin/crossfire

# Windows (PowerShell)
setx PATH "%PATH%;C:\Path\To\CrossFireL"

Language Package Setup

  • Python: Install Python 3.9+ from python.org, then pip is included.
  • NodeJS: Install NodeJS (16+) from nodejs.org, npm comes bundled.
  • Rust: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Ruby: sudo apt install ruby-full (Linux), brew install ruby (macOS), Windows Installer
  • PHP: sudo apt install php-cli composer or Composer installer
  • Java: Install OpenJDK 11+ via sudo apt install openjdk-11-jdk or Adoptium
  • C++: Install vcpkg: git clone https://github.com/microsoft/vcpkg.git && ./bootstrap-vcpkg.sh
  • Swift: Install Swift from swift.org
  • Go: Install Go 1.18+ from golang.org
  • R: Install R from CRAN

System Package Managers:

  • Linux: apt, dnf, pacman, snap, flatpak
  • macOS: Homebrew, MacPorts
  • Windows: Chocolatey, Winget, Scoop

Usage

crossfire <package_name>
crossfire -um            # Update all managers
crossfire -um <manager>  # Update a specific manager
crossfire -up <package>  # Update a specific package

Arguments

  • <package_name> : Install or update the package. CrossFire auto-detects the manager.
  • -um, --update-manager : Update managers. Use ALL to update all supported managers.
  • -up, --update-package : Update a specific package using its manager.
  • -h, --help : Show help message and exit.

Updating Managers

# Update a specific manager
crossfire -um pip
crossfire -um npm

# Update all managers
crossfire -um

Examples

# Install a Python package
crossfire requests

# Install a NodeJS package
crossfire express

# Update a package
crossfire -up PyQt6

# Update all managers
crossfire -um

# Update a specific manager
crossfire -um npm

Notes

  • Cross-platform: Works on Linux, macOS, and Windows.
  • Automatic detection: No need to specify pip, npm, or other managers explicitly.
  • Extensible: Easily add more languages or managers in crossfire.py.
